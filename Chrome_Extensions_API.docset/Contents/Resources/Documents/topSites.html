chrome.topSites

Description: Use the chrome.topSites API to access the top sites (i.e. most visited sites) that are displayed on the new tab page. These do not include shortcuts customized by the user.
Availability: Since Chrome 38.
Permissions: "topSites"

Manifest

You must declare the "topSites" permission in your extension's manifest to use this API. 

      {
        "name": "My extension",
        ...
        "permissions": [
          "topSites",
        ],
        ...
      }

Examples

You can find samples of this API in Samples.
